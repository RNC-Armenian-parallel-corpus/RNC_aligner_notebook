# RNC_notebooks
Этот репозиторий содержит jupyter-блокноты и дополняющие их файлы.
- Блокнот `lingtrain_am_ru.ipynb` + `_exceptions.txt` + `news.zip`. С помощью блокнота можно выровнять парные тексты. Достаточно положить их в папку на уровне блокнота и следовать инструкции по запуску внутри этого блокнота.
Выравнивание осуществляется с помощью библиотеки [lingtrain-aligner](https://github.com/averkij/lingtrain-aligner). Также блокнот доступен в google colab: [Ссылка](https://colab.research.google.com/drive/1A2IhWYKdYOBvRxStCJaeW5Oi3y9-4lhe?usp=sharing). Тестовые файлы `news.zip` и `_exceptions.txt` – архив с парными текстамии и файл с исключениями соответственно. Чтобы протестировать блокнот на этих файлах, достаточно загрузить их в панель файлов и запустить все ячейки.
- Блокнот для извлечения текстов из XML на случай, когда исходные выровненные тексты были утеряны.
