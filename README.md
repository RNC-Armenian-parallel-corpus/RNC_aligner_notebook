# RNC_aligner_notebook
Этот репозиторий содержит блокнот, с помощью которого можно выровнять парные тексты. Достаточно положить их в папку на уровне блокнота и следовать инструкции по запуску внутри этого блокнота.
Выравнивание осуществляется с помощью библиотеки [lingtrain-aligner](https://github.com/averkij/lingtrain-aligner).

[Ссылка](https://colab.research.google.com/drive/1A2IhWYKdYOBvRxStCJaeW5Oi3y9-4lhe?usp=sharing) на блокнот в google colab.

В репозитории также находятся тестовые файлы -- архив с парными текстамии и файл с исключениями. Чтобы протестировать блокнот на этих файлах, достаточно загрузить их в панель файлов и запустить вес ячейки.
